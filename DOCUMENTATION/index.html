A) Theme Development Structure

Manvir is a Bootstrap 4 Personal One Page HTML Template and fully responsive personal, Manvir is a simple, modern, creative and responsive template. It will help you to presents yourself or your business resume more smartly and easily. This is built with HTML5, CSS3, Bootstrap4 and jQuery. It’s 100% responsive with clean codes. All components used in the template are well commented, w3 validated and are well documented.
